## Installation steps:

- Clone the Github repo: https://github.com/Sciederrick/sales_agent_dashboard

- Clone the Github repo: https://github.com/Sciederrick/sales-agent-dashboard-mock-server

- Move to the root directory

- For the frontend:

  - ```bash
    npm install
    ```

  - ```bash
    npm run dev
    ```

- For the backend:

  - ```bash
    pnpm install
    ```

  - ```bash
    npm start
    ```
